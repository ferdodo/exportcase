Usage Guidelines

When to use each component:

  • ec-typography: Wrap all text content to ensure consistent styling
  • ec-code: Display code snippets, commands, or technical content
  • ec-background: Apply once per page for visual foundation
  • ec-rules-title: Main title for rules documentation
  • ec-rule-content: Content area for rule descriptions
  • ec-example-group: Group examples with colored titles (good/bad)
  • ec-example: Container for individual examples with colored left border
  • ec-file-name: Display file names in examples
  • ec-explanation: Explanatory text for examples

Best Practices:

  1. Always wrap text content in ec-typography
  2. Use ec-code for any code-related content
  3. Use ec-example-group with green (#27ae60) for good examples and red (#e74c3c) for bad examples
  4. Use ec-example with matching colors for the left border
  5. Maintain consistent spacing between sections
  6. Test components in different contexts
